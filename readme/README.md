### how/why ✮⋆˙
I love using the pomodoro technique to manage my time while I'm studying and while I've found a lot of sites that have both a pomodoro timer and a to-do list, most were either limited by a paywall or weren't very visually pleasing. So, I decided to make my own tool! 

This is my second web development project, where I learned a lot about using JavaScript to make my site responsive to user input! Most of my time on this project was spent on the timer and tasks pages, where I worked on making the site interactive by allowing the user to set a timer of their choosing and enter/edit tasks. These two tabs also brought the most challenges because this was my first time using JavaScript with web development (as my first website had basically nothing to interact with). I found some helpful tutorials along the way and added a personal touch to them to make the website more fun!
